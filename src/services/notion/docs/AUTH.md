# Notion auth notes

The SDK uses a Notion integration token by default.

## Current setup

- Set `NOTION_API_KEY` or `NOTION_TOKEN`
- Optionally set `NOTION_VERSION`
- Share pages/databases with the integration if the workspace requires it

## Relevant docs

- Notion overview: https://developers.notion.com/guides/get-started/overview
- Notion SDK reference: https://github.com/makenotion/notion-sdk-js
